Condicionales

Los condicionales en Python te permiten tomar decisiones en función de una condición específica. Puedes controlar el flujo de tu programa y ejecutar diferentes bloques de código según se cumplan o no ciertas condiciones. A continuación, te explico cómo funcionan los condicionales en Python con ejemplos:

Estructura básica del condicional 'if':

El condicional 'if' se utiliza para ejecutar un bloque de código si una condición es verdadera. La estructura básica es la siguiente: 

      
      if condicion:
        # Bloque de código a ejecutar si la condición es verdadera

Hagamos un ejemplo incluyendo operadores de comparación:

      
      edad = 20

      if edad >= 18:
          print("Eres mayor de edad")

En este ejemplo, si la variable edad es mayor o igual a 18, se imprimirá el mensaje "Eres mayor de edad".

Condicionales if-else:

Puedes utilizar el condicional 'else' para ejecutar un bloque de código alternativo cuando la condición del 'if' sea falsa. La estructura es la siguiente: 

      
      if condicion:
        # Bloque de código a ejecutar si la condición es verdadera
      else:
        # Bloque de código a ejecutar si la condición es falsa

Hagamos un ejemplo tomando de referencia el ejemplo anterior:

      
      edad = 15

      if edad >= 18:
        print("Eres mayor de edad")
      else:
        print("Eres menor de edad")

En este ejemplo, si la variable 'edad' es mayor o igual a 18, se imprime "Eres mayor de edad". De lo contrario, se imprime "Eres menor de edad".

Condicionales if-elif-else:

Puedes utilizar múltiples condicionales 'elif' (abreviatura de "else if") para evaluar condiciones adicionales en secuencia y ejecutar el bloque de código correspondiente a la primera condición que sea verdadera. La estructura es la siguiente: 

      
      if condicion1:
        # Bloque de código a ejecutar si la condicion1 es verdadera
      elif condicion2:
        # Bloque de código a ejecutar si la condicion2 es verdadera
      else:
        # Bloque de código a ejecutar si ninguna de las condiciones anteriores es verdadera

Hagamos un ejemplo:

      
      hora = 14
  
      if hora < 12:
          print("Buenos días")
      elif hora < 18:
          print("Buenas tardes")
      else:
          print("Buenas noches")

En este ejemplo, si la variable 'hora' es menor que 12, se imprime "Buenos días". Si es mayor o igual a 12 pero menor que 18, se imprime "Buenas tardes". De lo contrario, se imprime "Buenas noches".

💡 Los condicionales te permiten tomar decisiones y controlar el flujo de ejecución de tu programa en función de condiciones específicas. Puedes combinar condicionales, utilizar operadores lógicos y comparaciones para crear condiciones más complejas. Estos son solo ejemplos básicos, pero los condicionales en Python son muy flexibles y te permiten construir estructuras de control más sofisticadas para adaptarsea tus necesidades.
